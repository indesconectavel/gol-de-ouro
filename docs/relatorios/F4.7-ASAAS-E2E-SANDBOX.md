## 11. Webhook oficial (requisito crítico)

O teste **não** fabrica payload manualmente. Fluxo:

1. Registra webhook efêmero via `POST /v3/webhooks` (apiVersion 3)
2. Sobe receptor HTTP local + tunnel público (cloudflared/localtunnel)
3. Valida reachability com POST de prova
4. Após `confirmSandboxPayment`, aguarda POST oficial do Asaas
5. Remove webhook efêmero ao final (`DELETE /v3/webhooks/{id}`)

URL pública alternativa: `ASAAS_E2E_WEBHOOK_PUBLIC_BASE` ou `ASAAS_WEBHOOK_TUNNEL_URL`.

**Tentativa 1 — Script via Base64 no container:**  
Codificar o script em Base64 e executar com:

`node -e "eval(Buffer.from(process.argv[2],'base64').toString())" _ <BASE64>`

No PowerShell, as aspas em torno de `'base64'` não foram preservadas até o processo remoto (o shell/Node recebeu `base64` como identificador), gerando `ReferenceError: base64 is not defined`. Uso de `String.fromCharCode(98,97,115,101,54,52)` para evitar aspas resultou em comando muito longo e em erro de ambiente (“host unavailable” / “malformed resolve command”).

**Tentativa 2 — One-liner Node sem Base64:**  
Comando único com `node -e "var t=process.env...."` passado em `-C`. O PowerShell interpretou trechos do comando (ex.: `!!t`, `'Bearer '+t`) e gerou erros de parsing antes de chamar o flyctl.

**Tentativa 3 — Comando lido de arquivo:**

O flyctl retornou **"Error: accepts at most 1 arg(s), received 3"**, indicando que o valor de `-C` foi interpretado como mais de um argumento (provável divisão por espaços no parsing do flyctl no Windows).

**Resultado:** Não foi possível executar o script **dentro do container** nesta sessão devido a limitações de quoting/parsing (PowerShell + flyctl `-C` no Windows).

## 4) Script para execução manual no container

Para obter a prova em ambiente Linux (bash) ou quando o comando puder ser passado sem ser quebrado pelo PowerShell, use no container:

```javascript
const https = require('https');
const token = process.env.MERCADOPAGO_DEPOSIT_ACCESS_TOKEN || '';
const id = '148697499270';
console.log('ENV_TOKEN_PRESENT', !!token, 'TOKEN_LEN', token ? token.length : 0);
const opts = {
  hostname: 'api.mercadopago.com',
  path: '/v1/payments/' + id,
  method: 'GET',
  headers: { Authorization: 'Bearer ' + token }
};
const req = https.request(opts, (res) => {
  let body = '';
  res.on('data', (c) => body += c);
  res.on('end', () => {
    console.log('HTTP_STATUS', res.statusCode);
    try {
      const j = JSON.parse(body);
      console.log('MP_JSON', JSON.stringify({
        id: j.id,
        status: j.status,
        status_detail: j.status_detail,
        transaction_amount: j.transaction_amount,
        date_created: j.date_created,
        date_approved: j.date_approved,
        external_reference: j.external_reference,
        payer_email: j.payer && j.payer.email ? j.payer.email : null
      }, null, 2));
    } catch (e) {
      console.log('RAW_BODY_HEAD', String(body).slice(0, 500));
    }
  });
});
req.on('error', (e) => console.log('REQ_ERROR', e.message));
req.end();
```

Exemplo de execução no container (bash): salvar em `/tmp/q.js` e rodar `node /tmp/q.js`, ou codificar em Base64 e usar `node -e "eval(Buffer.from(process.argv[1],'base64').toString())" <BASE64>`.

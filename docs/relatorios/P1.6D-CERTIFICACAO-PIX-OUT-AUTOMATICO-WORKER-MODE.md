## FASE 1 — Auditoria do Fluxo Automático

### Cadeia ponta a ponta (homologada)

```text
POST /api/withdraw/request (app)
  → validação PixValidator + débito wallet + insert saques status pendente
  → ledger saque + taxa

Fly process group payout_worker
  → payout-worker.js setInterval(30s)
  → processPendingWithdrawals()
      gates: PAYOUT_PIX_ENABLED, PAYOUT_MODE≠manual, PAYOUT_AUTO_FROM_AT válido
  → query saques pendente/pending, created_at >= cut, limit 1 FIFO
  → processSingleWithdrawalPayout(source: 'worker')
      lock pendente → processando
  → createPixWithdrawCompat()
  → AsaasPayoutProvider.requestPixPayout()
  → asaas-http-client.createPixTransfer (POST /v3/transfers)
  → status aguardando_confirmacao

Webhook Asaas (app process)
  → POST /webhooks/asaas/transfer-validation (autorização)
  → POST /webhooks/asaas (TRANSFER_DONE)
  → processAsaasTransferWebhook
  → ledger payout_confirmado + saque processado
```

### Bloqueios identificados (pré-correção)

| Bloqueio | Impacto | Resolvido em P1.6D? |
|----------|---------|:-------------------:|
| Máquina worker `stopped` | Fila não processada | ✅ |
| `PAYOUT_MODE=manual` | Worker retorna `manualMode: true` | ✅ → `auto` |
| `restart.policy=on-failure` | Não reinicia após exit 0 | ✅ → `always` |
| Worker sem autostart pós-deploy | Permanece stopped até `fly machine start` | ✅ (restart + deploy) |

**Nota:** `approve-and-send` **ignora** `PAYOUT_MODE=manual` — por isso P1.7 homologou via admin enquanto worker estava inoperante.

## FASE 9 — Rollback

### Desativar PIX OUT automático (contingência)

```bash
# 1. Voltar modo manual (worker deixa de processar fila)
fly secrets set PAYOUT_MODE=manual -a goldeouro-backend-v2

# 2. Opcional: parar worker
fly machine stop d894655b5114e8 -a goldeouro-backend-v2

# 3. Opcional: desligar worker por env
fly secrets set ENABLE_PIX_PAYOUT_WORKER=false -a goldeouro-backend-v2

# 4. Gates PIX OUT (extremo — afeta também admin approve-and-send)
# ASAAS_PIX_OUT_PRODUCTION_ENABLED=false etc. (ver ASAAS.PIPELINE.5 rollback)
```

### Reativar após rollback

```bash
fly secrets set PAYOUT_MODE=auto PAYOUT_AUTO_FROM_AT=<ISO-8601> -a goldeouro-backend-v2
fly machine start d894655b5114e8 -a goldeouro-backend-v2
```

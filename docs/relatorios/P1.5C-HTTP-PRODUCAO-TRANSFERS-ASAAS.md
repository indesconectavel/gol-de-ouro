## 3. Wiring HTTP

```text
createPixWithdraw (produção, gates ON)
    ↓
guardAsaasPixOutProduction() → null
    ↓
createPixTransfer({ httpGate: 'pixOutProduction' })
    ↓
asaasApiRequest → isAsaasPixOutProductionHttpEnabled()
    ↓
POST https://api.asaas.com/v3/transfers
```

`isAsaasPixOutProductionHttpWired()`:

```javascript
return isAsaasPixOutProductionConfigured();
```

Sem duplicação: wired = configured (ready + `ASAAS_PIX_OUT_PRODUCTION_ENABLED` + `PAYMENT_ENGINE_PIXOUT_ENABLED`).

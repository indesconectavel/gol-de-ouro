## Resultado final

# FAIL

**Interrupção da cadeia:** `Payment Engine` → `claimAndCreditApprovedPixDeposit` → RPC Supabase `claim_and_credit_approved_pix_deposit` → **`invalid_mercadopago_id`**

O pagamento foi **confirmado no Asaas**, o webhook **chegou e foi aceito** pelo engine (HTTP 200), mas o **crédito em wallet/ledger não ocorreu** porque a RPC SQL só aceita IDs numéricos (`^\d+$`, formato Mercado Pago) e **rejeita** IDs Asaas (`pay_…`). O fallback JavaScript em `server-fly.js` **não é executado** quando a RPC retorna objeto com `ok: false`.

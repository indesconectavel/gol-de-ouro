## 8. GO-LIVE — Primeiro PIX REAL hoje?

## **NÃO**

### Impedimentos exatos (ordem de bloqueio)

1. **Conta Asaas produção não auditada** — sem chave `$aact_prod_` disponível; API produção retorna 401 com credencial sandbox.
2. **Credenciais produção não confirmadas no Fly** — `flyctl` sem login; não há evidência de `ASAAS_API_KEY` / `ASAAS_WEBHOOK_TOKEN` prod em runtime.
3. **`ASAAS_PRODUCTION_ENABLED=false`** — gate financeiro fechado (correto pré-go-live, mas impede PIX real).
4. **`PAYMENT_WEBHOOK_ENGINE_ENABLED=false`** em produção Fly — webhook Asaas retorna **404**; nenhum crédito processado.
5. **Webhook Asaas produção não verificado** no painel (URL, eventos, token, fila).
6. **PIX IN via backend não implementado para produção** — `createPixDeposit` cai em stub fora do modo sandbox HTTP.
7. **Nenhum teste de ciclo completo** depósito real → webhook → `pagamentos_pix` → saldo em produção (P1.0 §5 não executado).
8. **Provider efetivo ainda Mercado Pago** — arquitetura preparada, operação financeira atual não é Asaas.

## 9. VEREDITO P1.1A

# NOT READY

A conta Asaas **produção** não pôde ser auditada integralmente por **ausência de credenciais de produção** e **impossibilidade de verificar secrets Fly**. O backend possui **wire P1.0 implantado e protegido**, mas a **configuração operacional** (chaves, flags, webhooks, PIX IN API) **não está pronta** para o primeiro ciclo financeiro real via Asaas.

### Condições para reauditoria (P1.1B sugerido)

1. Operador autenticar `flyctl` e confirmar secrets: `ASAAS_API_KEY` (prod), `ASAAS_WEBHOOK_TOKEN`, flags §4 do checklist P1.0.
2. Fornecer acesso read-only à API produção (`$aact_prod_…`) **sem expor a chave em chat/commits**.
3. Executar probes read-only: `GET /myAccount`, `/finance/balance`, `/webhooks`, `/pix/addressKeys`, `/myAccount/status`.
4. Confirmar webhook no painel Asaas apontando para `https://goldeouro-backend-v2.fly.dev/webhooks/asaas` com eventos de pagamento.
5. Decidir estratégia PIX IN: implementar HTTP produção no backend **ou** aceitar cobrança manual inicial no painel Asaas.

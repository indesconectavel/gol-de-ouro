## 2. Schema Anterior

Migration V1-S (`20260424_mp_payout_transaction_intents.sql`) — orientada ao Mercado Pago:

| Coluna | Tipo | Função |
|--------|------|--------|
| `mp_transaction_intent_id` | TEXT | ID MP transaction-intents |
| `payout_external_reference` | TEXT | Referência externa (provider-agnostic) |
| `mp_payout_status` | TEXT | Status payout MP |
| `mp_payout_raw` | JSONB | Snapshot sanitizado MP |
| `last_mp_sync_at` | TIMESTAMPTZ | Última sync MP |

Índices legados: `saques_payout_external_reference_uidx`, `saques_mp_transaction_intent_id_idx`.

**Gap P1.5A:** ausência de colunas para identificador, status e payload bruto Asaas.

## 4. Campos Adicionados

| Coluna | Tipo | Nullable | Espelho MP | Descrição |
|--------|------|----------|------------|-----------|
| `asaas_transfer_id` | TEXT | ✅ | `mp_transaction_intent_id` | ID retornado em POST/GET `/v3/transfers` |
| `asaas_transfer_status` | TEXT | ✅ | `mp_payout_status` | Status da transferência Asaas |
| `asaas_payout_raw` | JSONB | ✅ | `mp_payout_raw` | Snapshot sanitizado (auditoria) |
| `last_asaas_sync_at` | TIMESTAMPTZ | ✅ | `last_mp_sync_at` | Última sincronização com API Asaas |

**Índices novos (parciais, `IF NOT EXISTS`):**

- `saques_asaas_transfer_id_idx` — `(asaas_transfer_id) WHERE asaas_transfer_id IS NOT NULL`
- `saques_asaas_transfer_status_idx` — `(asaas_transfer_status) WHERE asaas_transfer_status IS NOT NULL`

`payout_external_reference` permanece **compartilhado** entre providers (já existente, unique parcial).

## 6. Migration

### Forward

**Arquivo:** `database/migrations/20260628_p15d_asaas_payout_transfer_columns.sql`

Características:

- `ADD COLUMN IF NOT EXISTS` — idempotente
- `CREATE INDEX IF NOT EXISTS` — idempotente
- `COMMENT ON COLUMN` — documentação auditável
- **Proibido no forward:** DROP, ALTER COLUMN, UPDATE, DELETE

### Rollback

**Arquivo:** `database/migrations/20260628_p15d_asaas_payout_transfer_columns_rollback.sql`

- `DROP INDEX IF EXISTS` (índices P1.5D)
- `DROP COLUMN IF EXISTS` (4 colunas Asaas)
- **Não toca** colunas `mp_*` nem `payout_external_reference`

### Aplicação (produção)

```sql
-- Forward (Supabase SQL Editor ou psql com DATABASE_URL)
\i database/migrations/20260628_p15d_asaas_payout_transfer_columns.sql
```

Requer aprovação operacional explícita antes de executar em produção (mesmo padrão V1-S).

```text
processSingleWithdrawalPayout → createPixWithdrawCompat (execução)
        ↑
server-fly.js / admin → injetavam createPixWithdraw (MP direto)
        ↑
Validação "createPixWithdraw ausente" — bypass arquitetural
```

| Ponto | Problema |
|-------|----------|
| `server-fly.js` | `require('./services/pix-mercado-pago').createPixWithdraw` + injeção |
| `adminWithdrawController.js` | Import + pass-through MP |
| `payout-worker.js` | Passava compat mas redundante |
| `processPendingWithdrawals` | Parâmetro `createPixWithdraw` obrigatório (não usado na execução P1.5E) |

| `controllers/adminWithdrawController.js` L3 | `pix-mercado-pago` import | Admin approve/send |
| `controllers/adminWithdrawController.js` L237 | `createPixWithdraw` param | Admin approve/send |

| `processPendingWithdrawals.js` | `createPixWithdraw` param | Validação only |

**Mantido (não é bypass de execução):**

| Arquivo | Referência | Motivo |

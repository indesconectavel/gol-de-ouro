|----------|-------|---------|
| `ASAAS_PIX_OUT_ENABLED` | Feature PIX OUT (existente) | `false` |

| **`ASAAS_PIX_OUT_PRODUCTION_ENABLED`** | **Gate financeiro PIX OUT produção** | **`false`** |
| **`PAYMENT_ENGINE_PIXOUT_ENABLED`** | **Engine PIX OUT Asaas** | **`false`** |

### Funções de gate

| Função | Descrição |
|--------|-----------|
| `isAsaasPixOutHttpEnabled()` | Sandbox HTTP (inalterado) |
| `isAsaasPixOutProductionHttpReady()` | Prontidão técnica prod (env + API key) |
| `isAsaasPixOutProductionConfigured()` | Ready + gates financeiros ON |
| `isAsaasPixOutProductionHttpWired()` | **`false`** até P1.5C |
| `isAsaasPixOutProductionHttpEnabled()` | Configured + wired → HTTP permitido |
| `resolveAsaasPixOutHttpGate()` | `'pixOut'` \| `'pixOutProduction'` \| `null` |
| `guardAsaasPixOutProduction()` | Guard explícito com código de erro |
| `getAsaasPixOutProductionBlockReason()` | Diagnóstico read-only |

### Códigos de erro (guard)

| Código | Quando |
|--------|--------|
| `ASAAS_PIX_OUT_DISABLED` | Feature off |
| `ASAAS_PIX_OUT_PRODUCTION_DISABLED` | Gate financeiro off |
| `PAYMENT_ENGINE_PIXOUT_DISABLED` | Engine off |
| `ASAAS_PIX_OUT_NOT_CONFIGURED` | API key ausente |
| `ASAAS_PIX_OUT_PRODUCTION_HTTP_NOT_WIRED` | Gates ON mas P1.5C pendente |
| `ASAAS_PIX_OUT_PRODUCTION_HTTP_DISABLED` | HTTP client bloqueado |

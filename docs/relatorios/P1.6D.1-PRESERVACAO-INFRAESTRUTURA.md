## FASE 7 — Rollback

### Reverter infraestrutura ao modo manual (pré-P1.6D automático)

```bash
# 1. Modo manual — worker não processa fila
fly secrets set PAYOUT_MODE=manual -a goldeouro-backend-v2

# 2. Reverter restart policy da máquina worker (opcional)
fly machine update d894655b5114e8 --restart on-failure --yes -a goldeouro-backend-v2

# 3. Parar worker (opcional)
fly machine stop d894655b5114e8 -a goldeouro-backend-v2

# 4. Reverter fly.toml no Git (revert do commit P1.6D.1) e redeploy
git revert <hash-p16d-1>
fly deploy -a goldeouro-backend-v2
```

### Reativar infraestrutura homologada

```bash
fly secrets set PAYOUT_MODE=auto PAYOUT_AUTO_FROM_AT=2026-07-02T00:00:00.000Z -a goldeouro-backend-v2
fly machine update d894655b5114e8 --restart always --yes -a goldeouro-backend-v2
fly machine start d894655b5114e8 -a goldeouro-backend-v2
# fly.toml com [[restart]] always já versionado — fly deploy reproduz
```

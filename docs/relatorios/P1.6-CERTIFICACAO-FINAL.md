## 5. Fluxograma Final

```
Cliente → Saque → Wallet (reserva)
    → Admin aprova
    → Payment Engine → POST /v3/transfers
    → Asaas (transfer PENDING)
    → [~5s] POST /webhooks/asaas/transfer-validation
    → Validação → {"status":"APPROVED"}
    → Asaas executa PIX
    → POST /webhooks/asaas (TRANSFER_DONE)
    → Wallet + Ledger reconciliados
```

## 7. Próximos Passos (Fase 8 — Produção Controlada)

1. Configurar no painel Asaas Sandbox:
   - URL: `https://goldeouro-backend-v2.fly.dev/webhooks/asaas/transfer-validation`

2. Solicitar desativação autorização crítica (account manager)
3. `fly secrets set ASAAS_TRANSFER_AUTH_ENABLED=true`
4. Janela operacional: 1 saque valor mínimo
5. Validar logs `[ASAAS AUTH] authorized=true`
6. Confirmar TRANSFER_DONE + comprovante

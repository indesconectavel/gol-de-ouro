# 12 — Videos

**Prioridade:** P1 (apoio visual e contingência)  
**Finalidade:** gameplay, walkthrough executivo e gravações da demo.

---

## O que deve ficar aqui

| Vídeo | Conteúdo sugerido |
|-------|-------------------|
| **Gameplay** | Partida curta, UX do jogo |
| **Walkthrough** | Fluxo investidor: login → saldo → jogo → (opcional) PIX demo |
| **Demo gravada** | Ensaio completo da apresentação (mesmo roteiro que `04-DEMO`) |
| **B-roll** | Admin, métricas, /meta em tela (sem secrets) |

Formatos: MP4 (H.264), nomes com data `YYYY-MM-DD-descricao.mp4`.

---

## Uso na reunião

- Abertura opcional com clip curto (30–60 s)
- Fallback total se produção indisponível
- Envio pós-reunião para investidores que não puderam assistir ao vivo
